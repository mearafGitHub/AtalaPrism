## How To Run The API

Place the PRISM_SDK_PASSWORD in a .env file on the root of the repository. Then run

            docker-compose up

You can also build the image directly

            docker build --build-arg PRISM_SDK_PASSWORD=$PRISM_SDK_PASSWORD .
